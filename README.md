# Harry Potter Currency PWA

Готовая PWA для GitHub Pages.

## Что это
Калькулятор валют мира Гарри Поттера:
- галлеоны
- сикли
- кнаты

Есть:
- пополнение
- списание
- установка баланса
- автосохранение
- история операций
- установка на главный экран как PWA

## Как запустить через GitHub Pages

1. Создай новый репозиторий на GitHub
2. Загрузи в корень репозитория все файлы из этой папки
3. Открой Settings -> Pages
4. В разделе Build and deployment выбери:
   - Source: Deploy from a branch
   - Branch: main
   - Folder: / (root)
5. Сохрани настройки
6. Подожди, пока GitHub Pages выдаст ссылку на сайт

## Как установить на телефон

1. Открой сайт в Chrome на Android
2. Нажми меню браузера
3. Выбери "Добавить на главный экран"
